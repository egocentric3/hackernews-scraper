# Hacker News Scraper

Scrapes the Hacker News homepage using Playwright and exports article titles and links to a CSV file.

## Features

* Scrapes the latest Hacker News homepage
* Extracts article titles and URLs
* Exports results to a CSV file
* Uses Playwright for browser automation

## Requirements

* Python 3
* playwright

## Installation

```bash
pip install -r requirements.txt
playwright install
```

## Usage

```bash
python hackernews_scraper.py
```

Output file will be named `hacker_news.csv`

## Note

Built as a Python web scraping practice project using Playwright.
